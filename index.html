Intro

Machine learning fairness is a growing area of research, as ML tools are increasingly used for decision-making in contexts that affect everyday people. While machine learning algorithms might lack the types of biases that humans can explicitly or implicitly have, they can still have biased results in a number of ways, which are not always easy to understand in complex models.

Because of this, there are naturally efforts to create mathematical definitions of fairness that can be used to estimate if an algorithm is being fair. But, like how there are so many philosophical definitions of fairness, there are also many possible mathematical definitions of fairness.

Furthermore, even the choice of model to use is also a difficult decision we know little about. Some machine learning models may be better than others depending on what type of data your dataset has, but we do not have this kind of same understanding with fairness. All of this leads up to our research question:

are there some machine learning models that are more suited for fairness than others?

To study this question, we plan to do empirical analysis of different machine learning modelsâ€™ performance using a host of different fairness metrics. We will try to use models, metrics, and datasets that come from different real-world domains, as well as different types of prediction, and we will be to produce a 3-dimensional matrix of the performance of a certain machine learning model, for a certain definition of fairness, for a certain given dataset.
